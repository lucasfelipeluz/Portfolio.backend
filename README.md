## 📖 Docs
The API is composed of 10 private and 1 public requests:

Private Routes: 
- **GET|POST|PUT|DELETE - /projects:** List, Add, Update and Remove the Project data;
- **GET|POST|PUT|DELETE - /skills:** List, Add, Update and Remove the Skills data;
- **GET|PUT - /aboutme:** List and Update the About Me data.;

Public Routes: 
- **GET - /home:** List Projects, Skills and About Me data;
